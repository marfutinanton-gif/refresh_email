# refresh_email

HTML-шаблон для автоматической рассылки уведомлений о замене монитора (техрефреш).

## Файлы
- `techrefresh_email_outlook_safe.html` — основной шаблон письма.
- `assets/qr_techrefresh.png` — QR-код сообщества Техрефреш (для «богатого» CID-варианта).
- `preview.png` — превью отрендеренного письма.

## Плейсхолдеры
В тексте подставляются перед отправкой:
- `%NAME%` — имя компьютера пользователя;
- `%ADDRESS%` — адрес/расположение.

## Почему такая вёрстка
Письмо должно одинаково открываться в **классическом Outlook (движок Word)**, **Thunderbird**,
новом Outlook, веб- и мобильных клиентах, **во внутреннем сегменте без интернета**.

Из этого следуют ограничения:
- табличная вёрстка + инлайн-стили, сплошные цвета (без CSS-градиентов как основы);
- никаких `position/flex/transform/clip-path` — Word-движок их игнорирует;
- **никаких внешних или base64-картинок**: классический Outlook не показывает `data:`-картинки,
  а интернета во внутреннем сегменте нет. Поэтому монитор и логотип собраны из табличных блоков
  и текста, а QR заменён на кнопку-ссылку.

## Если нужен QR/логотип картинкой
Растровые картинки в классическом Outlook без интернета показываются только как
**CID-вложения** (картинка прикладывается к письму, в HTML ссылка `src="cid:имя"`).
Это настраивается на стороне скрипта отправки. Файл `assets/qr_techrefresh.png` готов для этого.
